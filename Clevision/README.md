# Clevision Landing Page

## Project Description

A responsive landing page built with HTML, Tailwind CSS, and JavaScript. The project includes a responsive header with mobile navigation, a hero section, and a footer with contact information and social media links.

## Tech Stack

* HTML5
* Tailwind CSS
* JavaScript (Vanilla JS)

## Features

* Responsive navigation bar
* Mobile hamburger menu
* Hero section with call-to-action button
* Responsive image layout
* Footer with contact details and social links
* Mobile-friendly design

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/clevision-landing-page.git
```

2. Navigate to the project folder:

```bash
cd clevision-landing-page
```

3. Open `index.html` in your browser.

Alternatively, use a local server such as VS Code Live Server.
